MuOS Style theme options, there's also a variety of Vertical Styles grid icon packs and wallpapeds not depicted in these images, mix and match, try out them out :) 
Update 16 FEB 2026 onwards - A major version updtae of the theme is now available, please headover to the the releases page of this repo to get it. Further instructions are also available.
The themes are adaptable to other devices but would require testers

Custom made themes for the Anbernic RG34XXSP Device Mainly running on [stock MOD OS](https://github.com/cbepx-me/Anbernic-H700-RG-xx-StockOS-Modification) but can also work on stock unmodified OS.

A fellow redditor has detailed on how to make your own themes and how to apply them on stock os, Check it out [here](https://www.reddit.com/r/SBCGaming/s/lKrFbe7Lec).
 , the stock mod os on the other hand has a theme manager, so you can just use that to install the theme as a zip
